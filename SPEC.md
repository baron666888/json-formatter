# JSON 格式化工具规范

## 1. 项目概述

- **项目名称**: JSON Formatter
- **项目类型**: Web 工具类应用
- **核心功能**: 提供美观的 JSON 输入、格式化、验证和语法高亮展示
- **目标用户**: 开发者、数据分析人员

## 2. 视觉规范

### 2.1 整体风格
- **设计风格**: 现代极简 + 柔和渐变背景
- **配色方案**:
  - 背景: `#0f172a` (深蓝黑) 渐变至 `#1e293b`
  - 主色调: `#3b82f6` (明亮蓝)
  - 强调色: `#10b981` (翠绿，用于成功状态)
  - 错误色: `#ef4444` (红色)
  - 文本: `#f8fafc` (近白) / `#94a3b8` (次要灰)
  - 卡片: `rgba(30, 41, 59, 0.8)` 带毛玻璃效果
  - 边框: `rgba(148, 163, 184, 0.1)`

### 2.2 字体
- **标题字体**: "Outfit", sans-serif
- **正文字体**: "Inter", sans-serif
- **代码字体**: "JetBrains Mono", monospace

### 2.3 组件设计
- **输入区域**: 大文本框，深色背景，带圆角 `12px`
- **输出区域**: 语法高亮代码展示区
- **按钮**: 渐变蓝色按钮，悬停有发光效果
- **功能控制**: 紧凑的工具栏，包含格式化、压缩、清空等功能

## 3. 功能规范

### 3.1 核心功能
1. **JSON 输入**: 支持粘贴或输入原始 JSON
2. **格式化**: 自动美化 JSON，缩进 2 空格
3. **压缩**: 将 JSON 压缩为单行
4. **验证**: 检查 JSON 有效性，错误时显示行号和错误信息
5. **复制**: 一键复制格式化后的 JSON
6. **清空**: 清空输入和输出

### 3.2 交互细节
- 实时格式化（输入后延迟 300ms 自动格式化）
- 错误时输入框边框变红
- 成功格式化后输出区有淡入动画
- 复制成功显示 toast 提示

### 3.3 状态处理
- **空状态**: 显示引导文案 "粘贴或输入 JSON 开始格式化"
- **错误状态**: 显示错误信息和具体行号
- **成功状态**: 显示格式化后的高亮代码

## 4. 技术栈

- **框架**: Nuxt 4
- **样式**: CSS (自定义，不使用 Tailwind)
- **语法高亮**: 使用 highlight.js 或自定义 CSS 方案

- [x] 错误提示准确

## 6. 多语言支持

### 6.1 支持语言
- **English (en)** - 默认语言
- **中文 (zh)** - 简体中文
- **日本語 (ja)** - 日语

### 6.2 实现方式
- 使用自定义 `composables/useI18n.ts` 实现
- 翻译消息直接定义在 TypeScript 文件中
- 通过 `useState` 管理当前语言状态

### 6.3 翻译内容
所有用户界面文本均已国际化：
- 标题和副标题
- 按钮文本（格式化、压缩、复制、清空）
- 输入输出区域标签
- 错误提示信息
- Toast 通知消息
- JSON 嵌套对象/数组的计数标签（keys/items）
