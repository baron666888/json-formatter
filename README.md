# JSON Formatter

一个美观大方的 JSON 格式化工具，基于 Nuxt 4 构建。

## 功能特性

- ✨ JSON 格式化（美化）
- 🗜️ JSON 压缩
- ✅ JSON 验证（带错误行号提示）
- 📋 一键复制
- 🎨 语法高亮
- 🌙 深色主题设计

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用说明

1. 在左侧输入框粘贴或输入 JSON
2. 自动实时格式化，或点击 "Format" 按钮手动格式化
3. 使用 "Compress" 按钮压缩 JSON
4. 点击 "Copy" 复制格式化后的结果

## 技术栈

- Nuxt 4
- Vue 3
- CSS3
